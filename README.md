Parse `InterOp` folder from Illumina run to keep useful informations.

## Install

```sh
git clone git@github.com:guillaume-gricourt/HmnIllumina.git
cd HmnIllumina
make
```


## Use

```sh
./HmnIllumina \
    --input	<FOLDER> \
    --output <FILE>
```

## Test

```sh
make test
```
